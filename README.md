# Mage Studio Documentation

This repository hosts [`mage-engine`](https://github.com/MageStudio/Mage) documentation, built with Docsify.

For the latest version, please visit:
[https://docs.mage.studio](https://docs.mage.studio)

---

## Getting Started

### Install dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

Documentation will be available on [`localhost:3001`](http://localhost:3001).

### Build for production

```bash
npm run build
```

This copies the docs folder to `dist/` for deployment.

### Preview production build

```bash
npm run preview
```
